# Property View

RESTful сервис для управления отелями с возможностью поиска, добавления удобств и получения статистики.

## Технологии

- **Java 21**
- **Spring Boot 4.0.5**
- **Spring Data JPA**
- **Liquibase** (миграции)
- **H2 Database** (in-memory)
- **SpringDoc OpenAPI** (Swagger UI)
- **Maven**

## Запуск

```bash
mvn spring-boot:run
```

Запуститься сервер по адресу localhost:8092
